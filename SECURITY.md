# Segurança

Se você encontrar uma vulnerabilidade de segurança, por favor priorize a comunicação responsável.

Opções para reportar:
- Abra uma *issue* no GitHub e marque-a como `security` (não publique detalhes sensíveis).
- Para informações sensíveis que não devem ser públicas, entre em contato via LinkedIn e marque o assunto como "Security report".

Responderemos e investigaremos o mais rápido possível.
